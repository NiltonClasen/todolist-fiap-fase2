# Aplicação de Lista de Tarefas

Esta é uma aplicação simples de Lista de Tarefas construída com Java, Spring Boot e MongoDB. A aplicação permite aos usuários criar, ler, atualizar e deletar tarefas.

## Pré-requisitos

- Java 21
- Maven 3.9.9
- Docker
- Docker Compose
